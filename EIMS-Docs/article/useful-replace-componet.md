# 拆包上传常用替代控件

用这些代码块替代拆不出来的代码块，把你的代码放进去。

## ea_ba

```html
<!--banner_con1-->
<div id="banner_con1" class="banner_con1" eims="xny">

</div>
<c_attribute r_database="" url=""></c_attribute>
<!--end_banner_con1-->
```

```css
/*ea_ba*/
/*banner_con1*/
.banner_con1 {
    width: 100%;
    height: 398px;
    background-image: url();
    background-position: center;
    background-size: cover
}
/*end_banner_con1*/
/*ea_ba*/
```
